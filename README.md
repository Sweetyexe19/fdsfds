# YouTube Shop Bot

Telegram-магазин YouTube-каналов на **aiogram 3** + **SQLite**.

## Возможности

- Главное меню: покупка, отзывы, корзина, поддержка, гарантии, соглашение
- Каталог категорий с количеством в наличии
- Корзина и выбор количества
- Оплата: **ЮKassa** (рубли) и **криптовалюта** (адреса задаются в админке)
- **ЮKassa** — автоматическая проверка и выдача (+ кнопка «Проверить оплату» через API)
- **Крипто** — «Проверить оплату» отправляет заявку админу на ручное подтверждение
- Админ-панель: категории, массовая загрузка TXT, настройки, ручная продажа
- Защита от двойной продажи (резерв + транзакции)
- **Скачать проданные** — архив `sold_archive.txt`, после скачивания очищается
- **Seed-фраза** — шифрование данных каналов в БД (Fernet + PBKDF2)

## Формат товара (1 строка = 1 канал)

```
логин:пароль:резервка:пароль_почты:ключ2фа:ссылка_на_канал
```

Пример:

```
user@gmail.com:Pass123:backup@mail.ru:BackupPass:JBSWY3DPEHPK3PXP:https://youtube.com/channel/UC...
```

## Установка

```bash
cd d:\pythonProjects\youtubeshop
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
```

Заполните `.env`:

| Переменная | Описание |
|------------|----------|
| `BOT_TOKEN` | Токен от [@BotFather](https://t.me/BotFather) |
| `ADMIN_IDS` | Telegram ID админов через запятую |
| `REVIEWS_URL` | Ссылка на группу с отзывами |
| `SUPPORT_USERNAME` | Username поддержки без @ |
| `YOOKASSA_SHOP_ID` / `YOOKASSA_SECRET_KEY` | [ЮKassa](https://yookassa.ru/) |
| Крипто-адреса | `/admin` → **Крипто-кошельки** (TRC20, BEP20, ERC20, Aptos, TON, Solana) |

## Запуск

```bash
python main.py
```

## Админ-панель

Команда `/admin` (доступна пользователям из `ADMIN_IDS`).

- **Категории** — создать/редактировать название, описание, цену
- **Загрузить TXT** — массовое добавление каналов
- **Пометить проданным** — вручную по ID товара
- **Заказы** — подтвердить оплату вручную
- **Настройки** — приветствие, ссылки, гарантии, соглашение
- **Статистика** — остатки, продажи, выручка
